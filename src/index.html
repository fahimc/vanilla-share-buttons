Share buttons

Made simple

One file, no dependencies, easy html syntax and customisable

Installation

Download and add the script. Click on the download button on the navigation to get the script.

<script type="text/javascript" src="js/share.js"></script>

Usage
Basic

Here is an example of adding a facebook button to a page. In this example the button will automatically get the url from the page.

Add the markup and specify 'facebook' in the 'data-share' attribute.

You need the container div and you can add as many buttons as you like within it.

below is a list to values you can use to change the type of button.

  • facebook
  • twitter
  • pinterest
  • linkedin
  • googleplus
  • whatsapp
  • email
<div data-share-container>
 <div data-share="facebook" ></div>
</div>
Output

Usage
Basic All Button

Here is an example of adding all buttons to a page. In this example the button will automatically get the url from the page.

<div data-share-container>
  <div data-share="facebook" data-share-title="true"></div>
  <div data-share="twitter" data-share-title="true"></div>
  <div data-share="pinterest" data-share-title="true"></div>
  <div data-share="linkedin" data-share-title="true"></div>
  <div data-share="googleplus" data-share-title="true"></div>
  <div data-share="email" data-share-title="true" data-share-subject="testing share" data-share-text="hello world"></div>
  <div data-share="whatsapp" data-share-title="true"></div>
</div>
Output

Usage
With title

You can show the label of the button using the following data attribute (data-share-title="true").

<div data-share-container>
  <div data-share="googleplus" data-share-title="true"></div>
</div>
Output

Usage
Custom URL

You can set the url for the share button by passing it in via the data-share-url attribute

<div data-share-container>
  <div data-share="pinterest" data-share-url="http://google.com" data-share-title="true"></div>
</div>
Output

Usage
Add Share message

Some buttons have the ability to add a share message. You can use the data-share-text attribute to add the text. 

<div data-share-container>
  <div data-share="twitter" data-share-text="hello world" data-share-title="true"></div>
  <div data-share="googleplus" data-share-text="hello world" data-share-title="true"></div>
</div>
Output

Usage
Email attributes

For the email button you can pass in the subject and the body content via data attributes, data-share-subject and data-share-text

<div data-share-container>
  <div data-share="email" data-share-subject="this is the subject" data-share-text="hello world" data-share-title="true"></div>
</div>
